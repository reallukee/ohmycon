# Installazione

1. Clonare la repository

2. Aprire un emulatore di terminale in modalità amministratore

3. Digitare:

    ```terminal
    install.vbs
    ```

4. Seguire la procedura



# Disinstallazione

1. Clonare la repository

2. Aprire un emulatore di terminale in modalità amministratore

3. Digitare:

    ```terminal
    uninstall.vbs
    ```

4. Seguire la procedura
